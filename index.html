Photopea Smart Object Integration

Instructions:

  1. click start using photopea if found it
  2. Click "choose file" to load PSD file.
  3. Click "Select Input Folder" to choose the folder containing the images you want to process.
  4. Click "Select Output Folder" to choose the folder where the processed images will be saved.
  5. Click "Process All Images" to automatically process all images in the input folder.
  6. Processed images are saved automatically in the selected output folder.
